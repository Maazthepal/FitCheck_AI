# FitCheck AI 👕✨

FitCheck AI is an AI-powered fashion analysis application that helps users evaluate and improve their outfits. Upload an outfit photo and receive an instant style report including outfit scoring, style detection, color harmony analysis, fit balance evaluation, proportion analysis, and personalized recommendations.

The application uses computer vision and AI-based analysis to break down different aspects of an outfit and present them through an interactive and visually engaging report.

## Features

* 📸 Upload outfit images for analysis
* 🎯 AI-generated Drip Score
* 👔 Fashion style detection
* 🎨 Color harmony analysis
* ⚖️ Outfit balance and proportion evaluation
* 💡 Personalized style recommendations
* 📊 Detailed visual report with animated insights
* 📱 Responsive design for desktop and mobile
* ✨ Modern UI built with Framer Motion animations

## Tech Stack
